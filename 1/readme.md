This assignment implements a single-layer perceptron from scratch using NumPy, with a step activation function and the standard perceptron weight update rule. The perceptron class is first tested on the AND and OR logic gates, both of which achieve 100% accuracy since they are linearly separable problems. Next, a random synthetic dataset of 300 samples is generated using make_classification with added noise (flip_y=0.15) and low class separation (class_sep=0.5), then split into train and test sets and standardized using StandardScaler. Two perceptron models are trained on this dataset with different epoch values — 5 epochs giving 56.67% accuracy and 15 epochs giving 66.67% accuracy — showing that increasing the number of training epochs improves the model's accuracy. This confirms that while a single-layer perceptron works perfectly on simple linearly separable data like AND/OR gates, its accuracy is limited on noisier, overlapping data, and more training helps but cannot fully overcome that limitation.
